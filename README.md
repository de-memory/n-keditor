# Laravar-admin NKeditor编辑器

![](https://laravel-admin.org/storage/2020/09/22/BRjsepWWKVc2zfQ0APjO55ZZhXDO0jo2WYodCe17.png)

## 依赖

  -    | 版本  
 ---- | ----- 
 php  | >=7.0.0 
 encore/laravel-admin  | >=~1.6


## 安装

### composer 安装

```
composer require de-memory/n-keditor
```

### 发布资源

```
php artisan vendor:publish --provider=Encore\NKeditor\NKeditorServiceProvider
```

### 添加配置,在/config/admin下面的extensions里面添加

```
'n-keditor' => [

    // 如果要关掉这个扩展，设置为false
    'enable' => true,

    /*
    |--------------------------------------------------------------------------
    | 指定上传文件的服务器地址
    |--------------------------------------------------------------------------
    */
    'uploadJson' => '',
    /*
    |--------------------------------------------------------------------------
    | 指定浏览远程图片的服务器地址
    |--------------------------------------------------------------------------
    */
    'fileManagerJson' => '',
    /*
    |--------------------------------------------------------------------------
    | 主题。默认block
    |--------------------------------------------------------------------------
    | "light"           精简版
    | "black"           雅黑主题
    | "grey"            深灰主题
    | "blue"            淡蓝主题
    */
    'themeType' => 'black',
    /*
    |--------------------------------------------------------------------------
    | true时显示输入辅助线。默认值: true
    |--------------------------------------------------------------------------
    */
    'showHelpGrid' => true,
    /*
    |--------------------------------------------------------------------------
    | true时显示浏览远程服务器按钮。默认值: false
    |--------------------------------------------------------------------------
    */
    'allowFileManager' => true,
    /*
    |--------------------------------------------------------------------------
    | true时显示图片上传按钮。默认值: true
    |--------------------------------------------------------------------------
    */
    'allowImageUpload' => false,
    /*
    |--------------------------------------------------------------------------
    | true时显示视音频上传按钮。默认值: true
    |--------------------------------------------------------------------------
    */
    'allowMediaUpload' => true,
    /*
    |--------------------------------------------------------------------------
    | 上传图片、Flash、视音频、文件时，支持添加别的参数一并传到服务器。
    |--------------------------------------------------------------------------
    */
    'extraFileUploadParams' => [
        "name" => 'de-memory',
    ],
]
```
